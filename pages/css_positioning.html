Tipos de Posicionamiento CSS

Static, Relative, Absolute, Fixed

Static

Es el posicionamiento por defecto

Sigue las reglas del posicionamiento html

Relative

Se posicionan los objetos relativamente a su posicion estatica

Cordinadas: Top, Right, Bottom, Left.

Left: 50px (Se mueve 50px a la derecha) de donde tendria que estar

Cuando se mueve, no afecta la posicion de los otros elementos. Mantiene el layout estatico

Absolute

Se posiciona en relacion al componente padre

que tenga posicion relativa

Afecta el orden de los demas elementos

Left: 50px (Se pone 50px a la izquierda del padre)

Fixed

Mantiene laa posicion. Util para navbar, o sidebar
