todo elemento se trata como si fuese una caja. width 100 expande hasta ocupar toda la pagina. border: solid: muestra un borde (interno) se expande para afuera. se pueden sobreescribir. border-top 0 border-width 120 80 60 40 va en el sentido de las agujas del reloj sino se especifica tamaño tendra el que ocupan sus componentes. ese tamaño se puede ampliar con padding margin: espacio entre ese componente y los demas todos juntos forman el box-model
