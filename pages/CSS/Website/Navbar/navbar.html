Navbar

Barra de navegacion

Para hacer una barra de navegacion primero empezamos escribiendo el contenido en nuestro documento HTML.

Empezamos con la etiqueta <nav>, que es el elemento semantico para crear un navbar

Como una barra de navegacion contiene generalmente una lista de links, tiene sentido utilizar una <ul> (Lista no ordenada) donde cada <li> (List item) sea un <a>(enlace)


Dentro del enlance, tenemos el atributo href, que nos permite definir donde queremos que nos lleve el enlace. Con el '#' le establecemos como destino la ubicacion de algun elemento con ese id (Que debemos crear luego)

Ahora para darle estilo vamos al CSS y seleccionamos el elemento nav y le damos algo de estilo:


  • height: La altura del navbar
  • background-color: El color de fondo
  • display: flex (para alinear los items)
  • align-items: center (Para centrar en el medio los links)
  • padding: 0 40px (Para poner un poco de relleno a los costados)

Luego le damos estilo a los links en si:


  • list-style: none (Para quitar la bola de adorno)
  • display: flex
  • gap: 20px (El espacio entre los elementos)
