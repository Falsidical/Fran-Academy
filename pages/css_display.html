Display

Block

Algunos elementos ocupan un bloque(linea) entera

Por ejemplo, h#, p, div, listas y forms

Inline

Solo ocupa el espacio que necesita, en ancho y altura

No se puede cambiar el ancho(width)

Ejemplos: span, anchors

Se puede el cambiar el display en CSS

Inline-Block

Se pueden poner al lado en una linea, y darles ancho y altura

Ejemplos: imagenes, casi?

None

Oculta el componente. Para el layout, es como si no existiera

Hay otra propiedad: Visibility: Hidden que respeta el layout
